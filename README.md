# Mortgage-Calculator
Simple mortgage calculator that allows the user to calculate monthly payments.

 
Input:\
-Home Price\
-Down Payment\
-Interest Rate\
-Loan Term (10, 20, 30 years)\
\
Output:\
-Loan Amount\
-Monthly payment (based on selected loan term)
